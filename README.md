### 🚀 Proyectos Destacados

#### 🍽️ [RestaurantePro - Gestión de Reservaciones](https://github.com/marchena290/RestaurantePro)
*Sistema Full-Stack diseñado para la gestión eficiente de negocios gastronómicos.*
- **Arquitectura:** Implementación de arquitectura de 3 capas (Controller, Service, Repository).
- **Seguridad:** Autenticación **JWT** y autorización basada en anotaciones `@PreAuthorize`.
- **Base de Datos:** Mapeo de 7 entidades relacionadas mediante **Hibernate/JPA**.

#### 🎫 [Ticketing System - API Modular](https://github.com/marchena290/tickets-backend)
*API profesional para la gestión de incidentes técnicos.*
- **Especialidad:** Implementación avanzada de **RBAC** (Control de Accesos Basado en Roles).
- **Lógica de Negocio:** Cálculo automatizado de tiempos de respuesta basados en **SLA** por categoría de cliente.
