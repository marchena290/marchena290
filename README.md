### 🚀 Proyectos Destacados

#### 🍽️ [Restaurante Pro — Sistema de Reservaciones](https://github.com/marchena290/restaurante_pro_backend)
*Proyecto de graduación universitaria · Java 21 · Spring Boot 3.5 · PostgreSQL*

- Arquitectura modular por capas aplicando principios SOLID y Clean Code.
- Seguridad con JWT, cifrado BCrypt y autorización granular mediante @PreAuthorize (RBAC).
- Esquema relacional en PostgreSQL bajo normalización 3FN.
- Documentación de API con Swagger/OpenAPI.

#### 🎫 [Enterprise ITSM API](https://github.com/marchena290/system-tickets_backend)
*Proyecto universitario · NestJS · TypeScript · TypeORM · PostgreSQL*

- API RESTful para gestión de servicios de TI desarrollada de forma individual.
- Motor de lógica de negocio para cálculo automático de SLAs (6h, 24h y 48h).
- Autenticación JWT con control de accesos por roles (Cliente, Soportista, Supervisor).

#### 🏨 [Restaurante Fénix — Sistema de Gestión Administrativo](https://github.com/marchena290/Restaurante-Managed-System-NE)
*Proyecto universitario · C# · ASP.NET MVC 4.8 · SQL Server · Entity Framework*

- Sistema administrativo para gestión de clientes, menú, mesas y reservaciones.
- Arquitectura en capas con Entity Framework y procedimientos almacenados.
- Validación de modelos con Data Annotations.
